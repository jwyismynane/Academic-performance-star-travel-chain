# Academic Performance Star Travel Chain

This project is a comprehensive academic performance management system that combines modern web technologies with efficient backend services.

## Project Structure

The project consists of two main components:

1. `cool-admin-vue-8.x/` - Frontend application built with Vue.js
2. `cool-admin-midway-8.x/` - Backend service built with Midway.js

## Features

- Modern and responsive user interface
- Efficient backend services
- Academic performance tracking
- User management
- Data visualization

## Getting Started

### Prerequisites

- Node.js
- npm or yarn
- Git

### Installation

1. Clone the repository
```bash
git clone https://github.com/jwyismynane/Academic-performance-star-travel-chain.git
```

2. Install frontend dependencies
```bash
cd cool-admin-vue-8.x
npm install
```

3. Install backend dependencies
```bash
cd cool-admin-midway-8.x
npm install
```

## Development

### Frontend
```bash
cd cool-admin-vue-8.x
npm run dev
```

### Backend
```bash
cd cool-admin-midway-8.x
npm run dev
```
